Hospital Bed Network Connectivity Devices

Designed and implemented the hardware and firmware necessary to create a wired (NIU) and wireless (WIU) TCP/IP gateway for Hill-Rom hospital beds. The devices were designed to be bed agnostic so that any Hill-Rom smart bed could be connected to the connectivity system.
